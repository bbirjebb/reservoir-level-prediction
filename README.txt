
This project predicts the reservoir levels of reservoir levels in Gujarat. 

R007_Machine_Learning_Project_.ipynb- python file for the code of the project\

Please upload all these files on the runtime before running the code:

Evapnovmarch.csv- data regarding evapo-transpiration rate
Guj_dec.csv- reservoir level data for the month of December
Guj_feb.csv- reservoir level data for the month of February
Guj_jan.csv- reservoir level data for the month of January
Guj_march.csv-reservoir level data for the month of March
Guj_nov.csv- reservoir level data for the month of November
Gujrain_dec.csv- rainfall data for December
Gujrain_feb.csv-rainfall data for February
Gujrain_jan.csv- rainfall data for January
Gujrain_march.csv- rainfall data for March
Gujrain_nov.csv-rainfall data for November
